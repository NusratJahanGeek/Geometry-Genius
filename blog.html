1. What is DOM and what is the purpose of DOM?

DOM stands for Document Object Model. DOM is a structured way for accessing HTML Documents and making it interactive. The structure, style, and content of a document can be changed using DOM. It is an object-oriented way of showcasing a website, where we can use Javascript to modify the necessary things.

2. How will you select HTML elements using DOM? Name the DOM methods.

We can select HTML elements in 5 ways using DOM. They are -

  1. getElementsbyTagName(): It is used to select the HTML element using their tags (e.g. h1, li, etc.)
  2. getElementbyId(): It is used to select the HTML element using the specific ID. Each element has an unique ID, so a unique item can be called using this.
  3. getElementsbyClassNames(): Elements can be selected using class names using this method. Generally, it is used to select multiple items at once.
  4. querySelectorAll(): It is used to select specific HTML tags all at once. (e.g. list item of specific classes can be selected altogether). It returns a Nodelist.
  5. querySelector(): This tag is used to return the first one. No matter how much items is selected, it'll always return the first item. It returns the first item of the Nodelist.

3. What is event bubble?

When an element of the DOM receives an event, the event propagates to its parent elements. Event bubble works from inward to outward, where it can keep finding its root element. We can use stopPropagation() command to stop event bubble.

4. What is a callback function and why will you use it?

A callback function is a function that is passed into another function as an argument. After a function is executed, the callback function is executed with that.
We use callback function to handle the functions better. When we need to execute one function only after the execution of another function is completed, we use callback function.
